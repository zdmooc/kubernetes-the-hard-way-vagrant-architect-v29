# Demo Script

## Pitch 1 minute
Ce dépôt montre Kubernetes The Hard Way, mais orienté architecte : composants, flux, diagnostics, audit.

## Démonstration 5 minutes
1. montrer la topologie ;
2. montrer les scripts ;
3. montrer les unités systemd ;
4. montrer les manifests ;
5. montrer les runbooks et la matrice d'audit.

## Démonstration 15 minutes
Ajouter :
- validation cluster ;
- scénarios d'incident ;
- recommandations d'architecture.
