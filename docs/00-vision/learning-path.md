# Learning path

## Niveau 1 — Voir
Comprendre la topologie, les composants et les flux.

## Niveau 2 — Construire
Bootstraper le cluster avec Vagrant, certificats, etcd, control plane, workers.

## Niveau 3 — Exploiter
Vérifier, diagnostiquer, faire des smoke tests.

## Niveau 4 — Auditer
Évaluer maturité, sécurité, résilience, gouvernance et recommandations.
