# Mission Readiness Checklist

Avant d'utiliser ce dépôt comme base de démonstration ou de mission :

- topologie Vagrant stable ;
- inventaire documenté ;
- versions explicites ;
- validation minimale reproductible ;
- rapport d'exécution présent ;
- schémas à jour ;
- scénarios de panne connus ;
- limites et hypothèses documentées.
