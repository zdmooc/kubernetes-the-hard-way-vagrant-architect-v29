# Networking

Concepts clés :
- IP de Pod,
- Service,
- EndpointSlice,
- CoreDNS,
- kube-proxy,
- CNI,
- Ingress / Gateway API,
- NetworkPolicy.

Un bon architecte doit pouvoir expliquer comment un Pod joint un autre Pod, comment un client trouve un Service et quel composant applique les politiques réseau.
