# Release notes V17

## Objectif
Faire de la V16 une version consolidée, cohérente et publiable.

## Corrections majeures
- README remis à niveau sur la topologie et le statut du dépôt.
- Numérotation ADR homogénéisée.
- Numérotation des labs homogénéisée, avec suppression des doublons.
- Ajout d’index pour les ADR et les labs.
- Renforcement des scripts cœur de bootstrap (`etcd`, `control plane`, `workers`).
- Centralisation des notes de release dans `docs/10-release/`.

## Résultat
La V17 est la première version qui cherche explicitement la cohérence éditoriale et structurelle, pas seulement l’ajout de nouveaux composants.
