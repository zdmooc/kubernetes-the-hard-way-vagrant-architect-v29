# Release Notes V10

## Statut
Version consolidée longue, pensée comme base sérieuse de publication publique et de démonstration.

## Ce qu'elle représente
- une trajectoire V6 → V10 continue ;
- un dépôt lisible ;
- une base de travail crédible pour aller vers une vraie automatisation complète.

## Ce qu'elle ne prétend pas être
- un produit fini 100 % automatisé ;
- un remplaçant direct d'une distribution Kubernetes packagée ;
- un framework supporté.
