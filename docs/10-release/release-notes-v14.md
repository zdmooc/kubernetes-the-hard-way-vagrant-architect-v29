# Release Notes V14

## Highlights
V14 introduces access governance and traceability:
- RBAC examples for viewers, namespace admins and cluster admins,
- ServiceAccount examples with explicit token-mount behavior,
- Pod Security Admission namespace labels,
- sample ValidatingAdmissionPolicy with CEL,
- kube-apiserver audit policy file,
- new bootstrap, runbook, audit checklist and ADR.

## Positioning
With V14, the repository no longer shows only how to build a cluster. It also starts to show how to govern it.
