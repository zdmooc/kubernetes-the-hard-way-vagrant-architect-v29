# Release Notes V15

## Main theme
Identity maturity and access model refinement.

## Added
- OIDC and identity architecture chapter,
- bootstrap chapter for federated identity preparation,
- diagnostics runbook for identity and OIDC,
- audit checklist for human vs workload identities,
